# CoffeeScript
CoffeeScript - это приложение с которым можно легко вызвать курьера, который принесёт ваше
горячее, вкусное, ароматное кофе, прямиком к вашей двери. Помимо горячего кофе, присутствует возможность заказать охлажденные напитки, в самое 
жаркое лето это в самый раз, а если вы больше предпочитаете чаи, то и они имеются в ассортименте приложения, и помимо всякий напитков, имеются мягкие, 
свежие, и самые вкусные выпечки кофейни. Опробовав приложение можно [установив](https://github.com/ryokoSya79/CoffeeScript/tree/master/APK) APK файл.

За основу дизайна и идеи взял с референса - Figma: https://www.figma.com/design/w1vnH7O6TpIJWQMwv59Q7P/Coffe-Shop-Mobile-app-UI-UX-Design--Community-?node-id=0-1&t=8LXVIjzRBTt2sI3k-1
